# Titus Kamunya — Portfolio Website

A personal portfolio website for **Titus Kamunya**, a software engineer and B2B technical writer based in Nairobi, Kenya. The site showcases his work history, expertise, education, and links to his published articles across the web.

## 📁 Project Structure

```
TK-Portfolio/
├── index.html      # Main HTML structure
├── styles.css      # All styling, animations, and responsive layout
├── main.js         # Scroll animations and contact form handler
└── README.md       # Project documentation
```

---

## ✨ Features

- Fully responsive design (mobile & desktop)
- Smooth scroll-triggered fade-in animations
- Work history timeline with links to published articles
- SEO optimized — meta tags, Open Graph, Twitter Card, and JSON-LD structured data
- Contact form with basic validation
- Dark theme with a clean, editorial aesthetic

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, grid, flexbox, keyframe animations)
- Vanilla JavaScript (IntersectionObserver API)
- Google Fonts — Syne, DM Serif Display, DM Sans

## 🚀 Getting Started Locally

No frameworks or build tools needed. Just clone and open:

```bash
git clone https://github.com/tituskamunya/Titus-Kamunya-Portfolio.git
cd Titus-Kamunya-Portfolio
```

Then open `index.html` in your browser, or use a local server:

```bash
# Python 3
python -m http.server 8000
```

Visit `http://localhost:8000` in your browser.
